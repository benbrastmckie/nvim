# Skill: Fix-It

Scan for FIX:/NOTE:/TODO:/QUESTION: tags in the codebase and create structured tasks with interactive selection and topic grouping.

## Contents

### Files

- `SKILL.md` - Skill definition and workflow

## Features

- Interactive tag scanning with preview
- Multi-select task creation
- Topic grouping for TODO and QUESTION items
- Dependency handling (learn-it -> fix-it)
- Content-based language detection for research tasks
